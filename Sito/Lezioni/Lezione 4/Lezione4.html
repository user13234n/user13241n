Contenuto della Lezione

Spiegazione degli Array Bidimensionali

Gli array bidimensionali, o matrici, sono una collezione di elementi organizzati in righe e colonne. In Python, le matrici possono essere rappresentate utilizzando liste annidate.

Utilizzo degli Array Bidimensionali

  • Le liste annidate sono utilizzate per memorizzare una sequenza di valori organizzati in righe e colonne.
  • Puoi accedere agli elementi di una matrice utilizzando indici bidimensionali.
  • Le matrici possono essere utilizzate per rappresentare dati strutturati, come griglie di gioco, tabelle di dati, o immagini.

Esempi di Utilizzo degli Array Bidimensionali

Creazione di una Matrice

# Creazione di una matrice 3x3
matrice = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

Accesso agli Elementi della Matrice

# Accesso agli elementi tramite indici
print(matrice[0][0])  # Output: 1
print(matrice[1][2])  # Output: 6

Modifica degli Elementi della Matrice

# Modifica di un elemento
matrice[1][1] = 10
print(matrice)  # Output: [[1, 2, 3], [4, 10, 6], [7, 8, 9]]

Inserimento di Valori Esterni

Puoi inserire valori in una matrice dall'esterno utilizzando l'input dell'utente:

matrice = []
for i in range(3):
    riga = []
    for j in range(3):
        valore = int(input(f"Inserisci un valore per la cella ({i},{j}): "))
        riga.append(valore)
    matrice.append(riga)

Generazione Casuale di Valori

Per generare valori casuali in una matrice, puoi utilizzare il modulo random:

import random

matrice_casuale = [[random.randint(1, 100) for _ in range(3)] for _ in range(3)]

Utilizzo in Programmi Più Lunghi

Le matrici sono comunemente utilizzate in programmi più lunghi per rappresentare dati strutturati, come griglie di gioco, tabelle di dati, o immagini, e per svolgere varie operazioni, come la manipolazione dei dati o l'elaborazione di immagini.

Introduzione a Python

Python è un linguaggio di programmazione ad alto livello, interpretato e dinamico, noto per la sua semplicità e leggibilità. È utilizzato in vari campi, dalla sviluppo web all'analisi dei dati, passando per l'intelligenza artificiale.

Installazione di Python

Per installare Python, visita il sito ufficiale python.org e scarica l'installer per il tuo sistema operativo. Segui le istruzioni per completare l'installazione.

Il primo programma in Python

Scriviamo il nostro primo programma in Python. Apri il tuo editor di testo preferito e inserisci il seguente codice:

print("Hello, World!")

Salva il file con l'estensione .py, ad esempio hello.py, ed eseguilo tramite il terminale/cmd:

python hello.py

Concetti di base

Variabili e tipi di dati

In Python, puoi dichiarare una variabile semplicemente assegnando un valore ad essa:

x = 5
y = "Hello, Python"
z = 3.14

Python supporta diversi tipi di dati, tra cui:

  • Interi: int
  • Stringhe: str
  • Numeri in virgola mobile: float
  • Liste: list
  • Dizionari: dict

Operatori

Python supporta vari operatori, come quelli aritmetici e di confronto:

# Operatori aritmetici
a = 10
b = 3
print(a + b)  # Output: 13
print(a - b)  # Output: 7
print(a * b)  # Output: 30
print(a / b)  # Output: 3.3333333333333335

# Operatori di confronto
print(a == b)  # Output: False
print(a != b)  # Output: True

Input e Output

Puoi ottenere input dall'utente utilizzando la funzione input():

nome = input("Inserisci il tuo nome: ")
print("Ciao, " + nome + "!")

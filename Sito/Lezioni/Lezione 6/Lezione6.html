Contenuto della Lezione

Spiegazione delle Funzioni

Le funzioni sono blocchi di codice riutilizzabili che eseguono una specifica operazione. In Python, le funzioni sono definite utilizzando la parola chiave `def` e possono accettare input, produrre output e/o modificare dati.

Utilizzo delle Funzioni

  • Le funzioni sono utilizzate per organizzare il codice in blocchi logici, migliorando la leggibilità e la manutenibilità del codice.
  • Le funzioni possono essere chiamate più volte all'interno di un programma, risparmiando tempo e sforzo nella scrittura del codice.
  • Le funzioni possono essere utilizzate per dividere un problema complesso in sottoproblemi più gestibili.

Perché Usare le Funzioni

Le funzioni sono utilizzate per promuovere la riutilizzabilità del codice, ridurre la duplicazione del codice e migliorare la manutenibilità del software. Rappresentano uno dei concetti fondamentali della programmazione modulare.

Esempi di Utilizzo delle Funzioni

Creazione di una Matrice

def crea_matrice(n, m, valore_iniziale=0):
    """
    Crea una matrice di dimensione n x m con un valore iniziale opzionale.
    """
    return [[valore_iniziale] * m for _ in range(n)]

# Esempio di utilizzo
matrice = crea_matrice(3, 3)
print(matrice)

Creazione di un Vettore

def crea_vettore(n, valore_iniziale=0):
    """
    Crea un vettore di dimensione n con un valore iniziale opzionale.
    """
    return [valore_iniziale] * n

# Esempio di utilizzo
vettore = crea_vettore(5)
print(vettore)

Trova Valori Maggiori di un Numero

def trova_valori_maggiori(vettore, soglia):
    """
    Restituisce una lista di valori maggiori di una soglia.
    """
    return [x for x in vettore if x > soglia]

# Esempio di utilizzo
valori = [10, 20, 30, 40, 50]
soglia = 25
risultato = trova_valori_maggiori(valori, soglia)
print(risultato)

def massimo(vettore):
    """
    Restituisce il valore massimo in un vettore.
    """
    return max(vettore)

def minimo(vettore):
    """
    Restituisce il valore minimo in un vettore.
    """

Calcolo della Media e della Mediana

def media(vettore):
    """
    Calcola la media dei valori in un vettore.
    """
    return sum(vettore) / len(vettore)

def mediana(vettore):
    """
    Calcola la mediana dei valori in un vettore.
    """
    ordinato = sorted(vettore)
    lunghezza = len(ordinato)
    if lunghezza % 2 == 0:
        mediano_basso = ordinato[lunghezza // 2 - 1]
        mediano_alto = ordinato[lunghezza // 2]
        return (mediano_basso + mediano_alto) / 2
    else:
        return ordinato[lunghezza // 2]

Risoluzione delle Equazioni

def risolvi_eq_1_grado(a, b):
    """
    Risolve un'equazione di primo grado nella forma ax + b = 0.
    Restituisce la soluzione.
    """
    if a == 0:
        if b == 0:
            return "L'equazione è indeterminata, ha infinite soluzioni"
        else:
            return "L'equazione è impossibile, non ha soluzioni"
    else:
        return -b / a
def risolvi_eq_2_grado(a, b, c):
            """
            Risolve un'equazione di secondo grado nella forma ax^2 + bx + c = 0.
            Restituisce le soluzioni reali.
            """
            delta = b**2 - 4*a*c
            if delta > 0:
                x1 = (-b + delta**0.5) / (2*a)
                x2 = (-b - delta**0.5) / (2*a)
                return x1, x2
            elif delta == 0:
                return -b / (2*a)
            else:
                return "L'equazione non ha soluzioni reali"

def calcola_integrale(f, a, b, n=100):
            """
            Calcola l'integrale definito di una funzione f(x) nell'intervallo [a, b] utilizzando il metodo dei rettangoli.
            Restituisce il valore approssimato dell'integrale.
            """
            dx = (b - a) / n
            integrale = 0
            for i in range(n):
                x = a + i * dx
                integrale += f(x) * dx
            return integrale
def moda(dati):
            """
            Calcola la moda di una lista di dati.
            Restituisce il valore più frequente.
            """
            conteggio = {}
            for dato in dati:
                conteggio[dato] = conteggio.get(dato, 0) + 1
            max_conteggio = max(conteggio.values())
            moda = [k for k, v in conteggio.items() if v == max_conteggio]
            return moda[0] if len(moda) == 1 else moda
def somma_vettori(v1, v2):
            """
            Calcola la somma di due vettori.
            Restituisce il vettore risultante.
            """
            if len(v1) != len(v2):
                return "I vettori devono avere la stessa lunghezza"
            return [x + y for x, y in zip(v1, v2)]
def prodotto_scalare(v1, v2):
            """
            Calcola il prodotto scalare di due vettori.
            Restituisce il valore scalare risultante.
            """
            if len(v1) != len(v2):
                return "I vettori devono avere la stessa lunghezza"
            return sum(x * y for x, y in zip(v1, v2))
def controllo_grandezza(matrice, righe, colonne):
            """
            Controlla la grandezza di una matrice.
            Restituisce True se la matrice ha le dimensioni specificate, False altrimenti.
            """
            return len(matrice) == righe and all(len(riga) == colonne for riga in matrice)

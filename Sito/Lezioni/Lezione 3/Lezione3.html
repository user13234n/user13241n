Contenuto della Lezione

Spiegazione degli Array Monodimensionali

Gli array monodimensionali, o vettori, sono una sequenza di elementi dello stesso tipo. In Python, gli array monodimensionali sono spesso rappresentati come liste.

Utilizzo degli Array Monodimensionali

  • Le liste sono utilizzate per memorizzare una sequenza di valori, come numeri, stringhe o altri oggetti.
  • Puoi accedere agli elementi di una lista utilizzando indici.
  • Le liste sono flessibili e possono essere modificate aggiungendo, rimuovendo o modificando gli elementi.

Esempi di Utilizzo degli Array Monodimensionali

Creazione di un Vettore

# Creazione di un vettore di numeri interi
vettore = [1, 2, 3, 4, 5]

# Creazione di un vettore di stringhe
nomi = ["Alice", "Bob", "Charlie"]

Accesso agli Elementi del Vettore

# Accesso agli elementi tramite indici
print(vettore[0])  # Output: 1
print(nomi[1])     # Output: Bob

Modifica degli Elementi del Vettore

# Modifica di un elemento
vettore[2] = 10
print(vettore)  # Output: [1, 2, 10, 4, 5]

Inserimento di Valori Esterni

Puoi inserire valori in un vettore dall'esterno utilizzando l'input dell'utente:

valore = int(input("Inserisci un numero: "))
vettore.append(valore)

Generazione Casuale di Valori

Per generare valori casuali in un vettore, puoi utilizzare il modulo random:

import random

vettore_casuale = [random.randint(1, 100) for _ in range(10)]

Utilizzo in Programmi Più Lunghi

I vettori sono spesso utilizzati in programmi più lunghi per memorizzare dati, effettuare calcoli e svolgere varie operazioni.

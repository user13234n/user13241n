Passaggio di Vettori e Matrici alle Funzioni

In Python, è possibile passare vettori e matrici come parametri alle funzioni. Questo consente di manipolare dati strutturati in modo efficiente.

Utilizzo dei Parametri Vettoriali e Matriciali

  • Le funzioni possono accettare vettori e matrici come parametri.
  • I parametri vettoriali possono essere utilizzati per operazioni su vettori, come somma, prodotto scalare, ecc.
  • I parametri matriciali possono essere utilizzati per operazioni su matrici, come somma, prodotto, trasposizione, ecc.

Esempi di Utilizzo

Calcolo della Somma di Due Vettori

def somma_vettori(v1, v2):
    """
    Calcola la somma di due vettori.
    Restituisce il vettore risultante.
    """
    if len(v1) != len(v2):
        return "I vettori devono avere la stessa lunghezza"
    return [x + y for x, y in zip(v1, v2)]

# Esempio di utilizzo
vettore1 = [1, 2, 3]
vettore2 = [4, 5, 6]
risultato = somma_vettori(vettore1, vettore2)
print(risultato)

Calcolo del Prodotto Scalare tra Due Vettori

def prodotto_scalare(v1, v2):
    """
    Calcola il prodotto scalare di due vettori.
    Restituisce il valore scalare risultante.
    """
    if len(v1) != len(v2):
        return "I vettori devono avere la stessa lunghezza"
    return sum(x * y for x, y in zip(v1, v2))

# Esempio di utilizzo
vettore1 = [1, 2, 3]
vettore2 = [4, 5, 6]
risultato = prodotto_scalare(vettore1, vettore2)
print(risultato)

Calcolo della Somma di Due Matrici

def somma_matrici(matrice1, matrice2):
            """
            Calcola la somma di due matrici.
            Restituisce la matrice risultante.
            """
            if len(matrice1) != len(matrice2) or len(matrice1[0]) != len(matrice2[0]):
                return "Le matrici devono avere le stesse dimensioni"
            return [[matrice1[i][j] + matrice2[i][j] for j in range(len(matrice1[0]))] for i in range(len(matrice1))]
        
        # Esempio di utilizzo
        matrice1 = [[1, 2], [3, 4]]
        matrice2 = [[5, 6], [7, 8]]
        risultato = somma_matrici(matrice1, matrice2)
        print(risultato)

Calcolo del Prodotto di Due Matrici

def prodotto_matrici(matrice1, matrice2):
            """
            Calcola il prodotto di due matrici.
            Restituisce la matrice risultante.
            """
            if len(matrice1[0]) != len(matrice2):
                return "Il numero di colonne della prima matrice deve essere uguale al numero di righe della seconda matrice"
            risultato = [[0] * len(matrice2[0]) for _ in range(len(matrice1))]
            for i in range(len(matrice1)):
                for j in range(len(matrice2[0])):
                    for k in range(len(matrice2)):
                        risultato[i][j] += matrice1[i][k] * matrice2[k][j]
            return risultato
        
        # Esempio di utilizzo
        matrice1 = [[1, 2], [3, 4]]
        matrice2 = [[5, 6], [7, 8]]
        risultato = prodotto_matrici(matrice1, matrice2)
        print(risultato)

Calcolo della Trasposta di una Matrice

def trasposta(matrice):
            """
            Calcola la trasposta di una matrice.
            Restituisce la matrice trasposta.
            """
            return [[matrice[j][i] for j in range(len(matrice))] for i in range(len(matrice[0]))]
        
        # Esempio di utilizzo
        matrice = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
        risultato = trasposta(matrice)
        print(risultato)
